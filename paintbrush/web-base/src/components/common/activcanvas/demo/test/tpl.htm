How To Improve The Rating

It appears that the image you selected is not able to be activated. This can happen for a few reasons. ActivCanvas works by using image recognition software which picks up unique points of contrast in an image and then ties this to audiovisual content. Sometimes if an image does not have enough points of contrast (ie. corners, strong lines, shapes) it is considered a ‘weak’ image for activation.

If your image has come back as unusable you can try uploading either another image of the same work, which may be higher resolution or of a better quality, or you can try ActivCanvas on another artwork entirely.
