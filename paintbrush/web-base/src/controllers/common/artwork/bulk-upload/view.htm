Adding Images

There are two methods of adding new images:

Drag & Drop

You can drag and drop multiple image files from a folder on your computer on to this page.

File Selection Dialog

Alternatively click on the Add Images(s) button, this will launch a file selection dialog which allows you to select one or more image files to upload.

Tips

When selecting files to upload via either method holding the Ctrl key then clicking on the files you want to upload will select just those files.

Holding the Shift key then clicking on two files will select both files and any files in between.

Minimum Required Data

In order to speed up the process of adding lots of images most artwork related data cannot be entered on this screen.

Apart from the images themselves only a title and type per artwork is required.

If your account is ActivCanvas enabled an optional Video field will be visible.

Removing Images

Click on the remove image button located at the top right of each image panel.
